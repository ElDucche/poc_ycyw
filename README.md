# POC de la fonctionnalité de chat pour Your_Car_Your_Way

Cette application montre la faisabilité de la fonctionnalité de chat synchrone entre client et support.
On a ici une vue support et une vue client activable par toggle.

## Installation
### Avec Docker
Dans un premier temps, cloner le projet en local :
`git clone https://github.com/ElDucche/poc_ycyw.git`

Puis lancer docker :
```bash
cd poc_ycyw
docker compose up
```

L'application est accessible sur le port 80 de votre [localhost](http://localhost:80)
